**With bullets**

    import {List} from '@parmenion/library'

    <List hasBullets>
      <li>This is a list item 1</li>
      <li>This is a list item 2</li>
      <li>This is a list item 3</li>
      <li>This is a list item 4</li>
      <li>This is a list item 5</li>
    </List>

**With spacing**

    import {List} from '@parmenion/library'

    </List>
